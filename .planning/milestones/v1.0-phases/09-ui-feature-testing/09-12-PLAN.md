<objective>
Fix the Playwright port mismatch that blocks all E2E tests from connecting to the running app.

Purpose: playwright.config.ts hardcodes port 3001 for both baseURL and webServerPort. The
Next.js dev server runs on port 3000. Every E2E test suite fails to connect because Playwright
targets the wrong port. This gap closure corrects both references and makes the port
configurable via PLAYWRIGHT_BASE_URL so no future manual edits are required when the port
changes.

Output: Updated apps/nextjs-app/playwright.config.ts with port 3000 as the default and
PLAYWRIGHT_BASE_URL / PLAYWRIGHT_PORT env-var overrides.
</objective>
