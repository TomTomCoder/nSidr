**1. Create `apps/nestjs-backend/src/features/agent/agent.controller.ts`**

Implement 6 endpoints under `/api/agent`:

```
POST   /api/agent                     — createAgent(dto, @Req user)
GET    /api/agent?baseId=X            — listAgents(@Query baseId)
GET    /api/agent/:id                 — getAgent(@Param id)
PATCH  /api/agent/:id                 — updateAgent(@Param id, dto, @Req user)
DELETE /api/agent/:id                 — removeAgent(@Param id)
POST   /api/agent/:id/run             — runAgent(@Param id, @Body ctx, @Res res) — SSE stream
PATCH  /api/agent/:id/tools/:toolName — toggleTool(@Param id, @Param toolName, @Body { isEnabled })
GET    /api/agent/:id/memories        — getMemories(@Param id)
```

For the `runAgent` endpoint, stream AgentRunEvents as NDJSON (SSE-compatible):
- Set headers: `Content-Type: text/event-stream`, `Transfer-Encoding: chunked`
- For each `AgentRunEvent` from `agentExecutionService.run(ctx)`: write `data: ${JSON.stringify(event)}\n\n`
- On completion or error: write `data: {"type":"done"}\n\n` and close

Use `@UseGuards(AuthGuard)` on all endpoints (same guard as ai.controller.ts).
Import `AgentService`, `AgentExecutionService`, `AgentMemoryService` from their respective files.

For `toggleTool`, use prismaService directly:
```typescript
await this.prismaService.agentTool.upsert({
  where: { agentId_toolName: { agentId: id, toolName } },
  update: { isEnabled },
  create: { agentId: id, toolName, isEnabled },
});
```

// Agent types
export const CreateAgentSchema = z.object({
  name: z.string().min(1).max(100),
  description: z.string().optional(),
  baseId: z.string(),
  instructions: z.string().optional(),
  modelKey: z.string().optional(),
  isPublic: z.boolean().optional().default(false),
});
export type ICreateAgent = z.infer<typeof CreateAgentSchema>;

export const UpdateAgentSchema = CreateAgentSchema.partial();
export type IUpdateAgent = z.infer<typeof UpdateAgentSchema>;

export const AgentSchema = z.object({
  id: z.string(),
  name: z.string(),
  description: z.string().nullable(),
  baseId: z.string(),
  instructions: z.string().nullable(),
  modelKey: z.string().nullable(),
  isPublic: z.boolean(),
  isActive: z.boolean(),
  maxIterations: z.number(),
  createdBy: z.string(),
  createdTime: z.string(),
});
export type IAgent = z.infer<typeof AgentSchema>;
