<objective>
Retrofit the largest direct-SDK consumer (D-15-06): `setting-open-api.service.ts` currently calls `createGateway` + `generateText` + `generateImage` inline across ~15 call sites with `@ai-sdk/openai` imports and direct factory use. Move all provider-instance construction into the gateway by injecting `AiService` (via `AI_SERVICE`) and building model instances through `aiService.getModelInstance()`. The `generateText`/`generateImage` calls themselves stay; only the inline `createGateway`/factory construction is removed.

Purpose: This is the highest-blast-radius retrofit — done in its own wave after the gateway resolver (Plan 01) is stable so no other slice depends on it.
Output: Gateway-routed open-api service with zero direct provider SDK imports + tests proving routing.
</objective>
