<objective>
Wave 1 (AGENT-05). Stand up the **Teable MCP server**: a new `agent/mcp/` module that exposes the existing agent tools (table/record CRUD, search_knowledge_base, etc.) as MCP **server** tools over StreamableHTTP using `@modelcontextprotocol/sdk` `Server`. Every tool invocation is RBAC-gated through authority-matrix as the scoped agent/user identity.

Purpose: External AI clients (Claude Desktop) AND Teable's own agent (17-02) consume one MCP layer.
Output: `agent/mcp/` server service + controller + tool registration, wired into agent.module.
</objective>
