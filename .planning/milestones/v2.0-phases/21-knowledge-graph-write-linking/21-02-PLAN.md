  <name>Task 1: Author KnowledgeDocService with createDoc + updateDoc</name>
  <files>apps/nestjs-backend/src/features/doc-search/knowledge-doc.service.ts</files>
  <behavior>
    - createDoc({spaceId, title, rawContent, createdBy, folderId?, sourceUrl?}) → returns {docId, status:'pending'}; inserts row with sourceType='agent', isIndexed=false, chunkCount=0, indexProgress=0, wordCount=computed
    - createDoc: throws NotFoundException when caller's space ≠ spaceId (RBAC; no enumeration)
    - createDoc: throws BadRequestException on empty title or empty rawContent (no silent defaults)
    - updateDoc({docId, rawContent, callerSpaceId, callerId}) → updates rawContent + wordCount; resets isIndexed=false, chunkCount=0, indexProgress=0 (D-21-04 full re-index); bumps updatedAt automatically
    - updateDoc: throws NotFoundException when doc.spaceId ≠ callerSpaceId (RBAC)
    - updateDoc: throws BadRequestException on empty rawContent
    - createDoc: also allows updating existing rows is NOT in scope — it strictly inserts
  </behavior>
  <action>
    Per D-21-01 and D-21-04 (decisions locked in 21-CONTEXT). Create
    `@Injectable() KnowledgeDocService` with constructor-injected `PrismaService`
    (value import — see critical_constraint #1).

    `createDoc(input: { spaceId; title; rawContent; createdBy; folderId?; sourceUrl?; })`:
    - Validate non-empty title + rawContent → throw `BadRequestException` if empty (per critical_constraint #3 no silent fallbacks)
    - Compute `wordCount = rawContent.trim().split(/\s+/).filter(Boolean).length`
    - `prisma.importedDoc.create({ data: { spaceId, title, sourceType: 'agent', rawContent, wordCount, createdBy, folderId, sourceUrl, isIndexed: false, indexProgress: 0, chunkCount: 0 } })`
    - Return `{ docId: row.id, status: 'pending' as const }`

    `updateDoc(input: { docId; rawContent; callerSpaceId; callerId; })`:
    - Validate rawContent non-empty
    - `const doc = await prisma.importedDoc.findUnique({ where:{ id:docId }, select:{ spaceId:true } })`
    - If `!doc || doc.spaceId !== callerSpaceId` → `throw new NotFoundException()` (RBAC opaque — no enumeration)
    - Compute new wordCount
    - `prisma.importedDoc.update({ where:{ id:docId }, data:{ rawContent, wordCount, isIndexed:false, indexProgress:0, chunkCount:0 } })` — the existing ingestion worker will re-pick it (D-21-04 full re-index, planner's discretion locked in 21-CONTEXT).
    - Return `{ docId, status: 'pending' as const }`

    Note: `prisma.docChunk.deleteMany({ where:{ docId } })` BEFORE the update — otherwise stale chunks remain searchable until the worker overwrites them. Wrap update+chunk-delete in `prisma.$transaction([...])` so partial failures don't leave inconsistent state.

    File header: `import { Injectable, BadRequestException, NotFoundException } from '@nestjs/common';` + value import of `PrismaService`.
  </action>
