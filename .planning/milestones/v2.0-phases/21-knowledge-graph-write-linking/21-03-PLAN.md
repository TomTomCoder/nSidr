  <behavior>
    - linkDocs({fromDocId,toDocId,label?,callerSpaceId,callerId}) → returns {linkId, fromDocId, toDocId, label}
    - linkDocs: throws BadRequestException when fromDocId === toDocId (per D-21-02 — fast fail before DB CHECK)
    - linkDocs: throws NotFoundException when either doc not in callerSpaceId (RBAC composes, planner discretion locked in 21-CONTEXT)
    - linkDocs: throws ConflictException on UNIQUE violation (label dedup — surface, do NOT silently swallow per critical_constraint #3)
    - getOutgoing({docId, callerSpaceId, limit=100}) → returns [{linkId, toDocId, toTitle, label, createdAt}], filtered by from-doc-in-space
    - getIncoming({docId, callerSpaceId, limit=100}) → returns [{linkId, fromDocId, fromTitle, label, createdAt}], filtered by to-doc-in-space
    - Both getters EXCLUDE legacy ingestion-extracted rows (`WHERE label IS NOT NULL`) so the UI panel shows only agent/user-authored semantic links — ingestion-extracted links are a separate concept rendered elsewhere
  </behavior>
  <action>
    `@Injectable() DocLinkService` — value-import `PrismaService` (critical_constraint #1).

    `linkDocs(input)`:
    1. If `input.fromDocId === input.toDocId` → throw `BadRequestException('self-link rejected')`
    2. RBAC scope check:
       ```ts
       const docs = await prisma.importedDoc.findMany({
         where: { id: { in: [input.fromDocId, input.toDocId] }, spaceId: input.callerSpaceId },
         select: { id: true },
       });
       if (docs.length < 2) throw new NotFoundException();
       ```
       (length<2 covers: one missing, one in different space, or both missing — opaque per RBAC convention)
    3. Insert. Catch Prisma `P2002` (unique violation on `doc_link_from_to_label_uq`) → throw `ConflictException('link already exists')`.
       ```ts
       try {
         const row = await prisma.docLink.create({
           data: {
             fromDocId: input.fromDocId,
             toDocId: input.toDocId,
             label: input.label ?? null,
             createdBy: input.callerId,
             linkText: '',          // legacy column, ingestion-owned
             linkType: 'internal',  // DocLinkType enum
           },
         });
         return { linkId: row.id, fromDocId: row.fromDocId, toDocId: row.toDocId!, label: row.label };
       } catch (err) {
         if (err && typeof err === 'object' && 'code' in err && (err as { code: string }).code === 'P2002') {
           throw new ConflictException('link already exists');
         }
         throw err;
       }
       ```

    `getOutgoing({ docId, callerSpaceId, limit = 100 })`:
    - RBAC: `const doc = await prisma.importedDoc.findUnique({ where:{ id:docId }, select:{ spaceId:true } })`; if missing or wrong space → `NotFoundException`
    - `prisma.docLink.findMany({ where:{ fromDocId:docId, label:{ not:null } }, take:limit, include:{ toDoc:{ select:{ id:true, title:true } } }, orderBy:{ createdAt:'desc' } })`
    - Map to `{ linkId, toDocId, toTitle, label, createdAt }`. Skip rows where `toDoc` is null (legacy external links — `toDocId` may be null).

    `getIncoming`: symmetric — `where:{ toDocId:docId, label:{ not:null } }`, `include:{ fromDoc:{ select:{ id, title } } }`.

    Imports: `import { Injectable, BadRequestException, NotFoundException, ConflictException } from '@nestjs/common';` + value-import PrismaService.
